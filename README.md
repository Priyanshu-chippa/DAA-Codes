# DAA Codes - Data Structures and Algorithms

This repository contains complete implementations of all Data Structures and Algorithms (DAA) topics including:

## Topics Covered

### Sorting Algorithms
- Bubble Sort
- Selection Sort
- Merge Sort
- Quick Sort (with different pivot selection strategies)

### Searching Algorithms
- Linear Search
- Binary Search (requires sorted array)

### Array Operations
- Finding Maximum and Minimum elements

### Dynamic Programming
- Knapsack Problem (Fractional and 0/1)
- Multistage Graph (Forward and Backward Approaches)

### Graph Algorithms
- MSP - Prim's Algorithm (Minimum Spanning Tree)
- Dijkstra's Algorithm (Single Source Shortest Path)
- Warshall's Algorithm (Transitive Closure)
- Floyd's Algorithm (All Pairs Shortest Path)
- Bellman-Ford Algorithm

## File Structure

Each algorithm is implemented in its own C file with:
- Simple and understandable variable names
- Logic comments explaining each step
- Console input/output for testing
- Time and space complexity analysis where applicable

## How to Use

1. Each `.c` file is a standalone program
2. Compile using: `gcc filename.c -o filename`
3. Run using: `./filename` (Linux/Mac) or `filename.exe` (Windows)
